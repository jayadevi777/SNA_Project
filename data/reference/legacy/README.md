# Legacy Reference Dataset

This folder contains the original fallback Tenkasi place dataset used before the live Google Places pipeline was fully configured.

Files here are kept for:

- fallback rendering when live processed data is missing
- comparison against earlier project iterations
- reference during report writing and dataset evolution

The main live project now relies on:

- `data/processed/places.csv`
- `data/processed/reviews.csv`
- `data/processed/edges.csv`
